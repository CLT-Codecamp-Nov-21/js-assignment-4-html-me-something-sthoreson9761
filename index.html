A New Way to Learn Java

What is "Javalin Warrior"?

Javalin Warrior is a web application designed for the purpose of teaching the basics of the Java programming language. The idea behind this game came forth during the discussion for my groups final project. After our discussion, we had come to an understanding that we all struggled a bit in our "Intro to Computer Programming 1" or ITSC-1212. When we first started at the University of North Carolina - Charlotte, we had little to no previous programming experience. The group decided to create this game in order for anyone new or interested in to computer programming to get a bit of experience with programming. We had decided on the Java programming language because it is one of the most commonly used programming languages in the Computer Science field. While Javalin Warrior is working and available to play, my group had a larger vision for what it should have been if we had more than a single semester to work on it.

The Larger Vision

The table bellow contains only a few of the features that we were unable to implement before the due date.

Feature Description
User profiles Contains the information for an inarticleidual user. This information consists of the inarticleiduals username and id, email, password, and levels they have completed and their input.
Level selection Contains the list of levels that the user has completed and re-attempt them.
User input flexibilty This is to allow the user to input a possibly correct answer instead of the exact line of code. Think of this as using unit testing only the expected output can only be true if it meets certain requirements.
Level difficulties This would allow the user to test their knowledge of programming. The lowest requires that the user input only this area of code, while the highest requires the user to write all their (Java) code for the level from scratch.

Just from the few items listed above, you can see how much of an undertaking this is.
